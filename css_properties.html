Position property

The position property of css specifies the how an element would be placed. If it is relative, the element is placed wrt to its original position. If it's absolute, then the element is positioned wrt to its parent element. If it's fixed, then the element is positioned wrt to its original position and moves along as the user scrolls down.

Display property

The display property of css specifies how an element is displayed in a webpage. If it is block, then no element can be displayed in the same line. If it's inline, then the element can have elements displayed next to it. If it's inline-block, then the element is displayed inline but its children are displayed block. Display set to none makes the element invisible.

Float property

The float property specifies how an element 'floats'. If it's set to left, the element 'floats' at the left side. Setting it to center, top, right and bottom makes the element 'float' center, top, right and bottom respectively.

Border property

The border property specifies a border that can be used to enclose an element. It is of the form: border-width(px|em|%) border-style(solid|dashed) border-color(any hexadecimal value corresponding to a color).

Color property

The color property specifies the color of the text contents in an html element, such as a div, body, etc.

Text-align property

This property specifies the alignment of the text. It can take values such as center, left, right which correspond to the literal meanings of the values.

Margin property

This property specifies the spacing surrounding an element. It consists of four parts: margin-top, margin-right, margin-bottom, margin-left all of which can be specified in px, em, or %.

Height property

This property specifies the height of the element. It can be stated in px, em, or %.

Width property

The width property specifies the width of the element. It can be stated in px, em, or %

Top property

The top property specifies the distance of the element from the top of the document, or the top of the parent element depending on the position of the element. If position is absolute, then the distance is from top of the parent element, and if the position is relative, distance is from the top of the document. It can be stated in px, em, or %

Left property

The left property specifies the distance of the element from the left edge of the document, or the left edge of the parent element depending on the position of the element. If position is absolute, then the distance is from top of the parent element, and if the position is relative, distance is from the top of the document. It can be stated in px, em, or %

Right property

The right property specifies the distance of the element from the right edge of the document, or the right edge of the parent element depending on the position of the element. If position is absolute, then the distance is from top of the parent element, and if the position is relative, distance is from the top of the document. It can be stated in px, em, or %

Padding property

The padding property specifies the spacing of a child element inside a parent element. It is a shorthand which consists of four properties: padding-top, padding-right, padding-bottom, padding-left. It can be stated in px, em or %.

Background color property

The background color property is used to specify the background color of an element. It's value can be either a hexadecimal code for the color, or the name of the color, or it can be RGB values.

Background image property

The background image property is used to specify the background image of an element. It's specified in the form of a url('') where the path of the image goes in between the quotes.

Background repeat property

The background repeat property is used to specify whether the background should repeat in the element. If the value is no-repeat, then the background doesn't repeat, if it's repeat-x, then the background repeats along the x-axis, and if the value is repeat-y, then the background repeats along the y-axis. Default is that the background repeats along the x and y axis.

Background position property

The background position property is used to specify the position of the background of an element. It can be specified either as top, left, center, left top, etc. or it can be specified as x,y co-ordinates which can be either in % or pixels.

Z-Index property

The z-index property specifies the index in the stack that the element holds. This property is used to control the visibility of the elements in case they overlap.

The transition property

The transition property is a shorthand for four transition properties: transition-property, transition-duration, transition-timing-function, and transition-delay. The transition-property specifies the css property for which the transition applies, transition-duration specifies for how long the transition happens (in seconds), transition-timing-function specifies the speed curve of the effect, while transition-delay specifies the time before the transition starts. Usually, the transition is triggered by hovering over an element.

Text transform property

The text-transform property is used to specify whether the text is uppercase, lowercase or capitalize.

Font family property

The font family property is used to specify the font style of the text.
